## 📌 Sobre o projeto

O Yoky é um bot para Discord desenvolvido com foco em **moderação e organização de servidores**.

Este repositório representa a **versão pública/demonstrativa** do projeto.
Existe também uma versão privada, que é onde o desenvolvimento principal e atualizações são mantidos.

---

## ⚙️ Funcionalidades

* 🧹 **Limpeza de chat** (`/clear`)
  Permite que administradores removam mensagens e mantenham o servidor organizado.

* 📊 **Registro de comandos** *(atualmente desativado)*
  Sistema que registrava comandos utilizados para auxiliar na detecção de comportamentos suspeitos.

* 🔌 **API interna (em desenvolvimento)**
  Estrutura criada para possíveis integrações futuras, como sistemas de recompensas baseados em dados do servidor.

---

## 🎯 Objetivo

O objetivo do Yoky é centralizar funções de moderação em um único bot, reduzindo a necessidade de múltiplas ferramentas dentro de um servidor Discord.

A ideia principal é facilitar a administração e melhorar a experiência de gerenciamento de comunidades.

---

## 🔒 Versão privada

A versão principal do Yoky é mantida de forma privada e recebe atualizações ocasionais.

Essa versão pública funciona apenas como **demonstração do projeto e de suas funcionalidades básicas**.

## 🧠 Observação
